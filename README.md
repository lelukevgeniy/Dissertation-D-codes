- "Tensor ML Decoding" - код мажоритарного декодера для тензорного произведения кодов Рида-Маллера

- "Is D-code Strong" - код зарегистрированной программы для оценки принадлежности D-кода к множеству стойких ключей криптосистемы типа Мак-Элиса

- "D-codes for Prob Decoders" - код программы, которая использовалась для подбора оптимальных D-кодов для использования в криптосистеме типа Мак-Элиса

- "McED Enc Dec Prob" - код программ для шифрования и расшифрования в криптосистеме типа Мак-элиса на основе D-кодов с использованием вероятностных декодеров ISDDecoder и MatrixDecoder
